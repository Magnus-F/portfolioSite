Appleseed

Appleseed was a project for my more advanced web design class, in which we were supposed to make a website using two APIs. I used a language API called RiTa and a weather API to make a massive mad-lib based news site. It randomly generates article titles on the left and when you click on one a radomly generated article will appear on the right.
